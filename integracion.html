3. Integración en Eclipse

Paso 1: Crear el Proyecto

Crea un Java Project normal.

⚠️
MUY IMPORTANTE: Si Eclipse te pregunta "Create module-info.java?", selecciona Don't Create. Si se crea, bórralo inmediatamente de la carpeta src.

Paso 2: Configurar Build Path

  1. Clic derecho en el proyecto -> Build Path -> Configure Build Path...
  2. Ve a la pestaña Libraries.
  3. Selecciona Classpath (Asegúrate de NO seleccionar Modulepath).
  4. Haz clic en el botón Add External JARs...
  5. Selecciona TODOS los archivos descargados (Core, Driver y Ocultos).
  6. Pulsa Apply and Close.
