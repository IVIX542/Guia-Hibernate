1. Requisitos Previos

Antes de empezar, asegúrate de tener instalado:

  • ✅ Eclipse IDE: Versión 2023 o superior (para Enterprise Java).
  • ✅ Java JDK: Versión 11, 17 o 21 (Recomendado 17).
  • ✅ MySQL Server: Puede ser mediante XAMPP, WAMP o instalado nativamente.
  • ✅ Cliente SQL: Workbench, DBeaver o phpMyAdmin.
